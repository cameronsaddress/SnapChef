# SnapChef Web App Archive

This directory contains the original Streamlit-based web application for SnapChef (v1.0.0-web).

## Archive Date
- Archived on: December 2024
- Last commit before archive: 4c1f2c0
- Git tag: v1.0.0-web

## Contents
- Full Streamlit web application
- Python backend with Grok-4 API integration
- Device fingerprinting system
- Social sharing features
- All assets and configurations

## Note
This web app has been archived to make way for the native iOS application development.
To run the web app, navigate to this directory and run:
```bash
streamlit run app.py
```
